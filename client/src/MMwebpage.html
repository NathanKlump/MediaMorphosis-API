MediaMorphosis-API

About

MediaMorphosis API, written in Rust, leverages FFmpeg for high-performance media conversion. It effortlessly transforms audio, video, and images into various formats, ensuring quality and speed. Ideal for developers needing a reliable, swift media converter.

Setup Details

Project Name: MediaMorphosis API The restricted users involved two user groups within the GCP VM: 'Admin' and 'Peasants'. The 'Admin' group has access to logs and system activities, while 'Peasants' are restricted from accessing logs to maintain data integrity. The restrictied resource involved video size... logging success rate is kinda a issue because the middleware reads the header bits and that has a chance of failing (even though it has not yet).
